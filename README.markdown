# ChronoWeaver

---

# 🇬🇧 English

## Overview

**ChronoWeaver** is a 2D time-manipulation platformer built with **Python** and **PyQt6**.

Players can manipulate time, create temporal knots, leave time trails, summon ghost versions of their past movements, collect energy crystals, defeat enemies, and overcome challenging platforming levels.

### Main Features

- Time manipulation gameplay
- Time trail visualization system
- Temporal knot creation
- Ghost replay mechanics
- Multiple enemy types
- Boss battle system
- Dynamic particle effects
- Dark and Light themes
- Multi-language support
  - English
  - Persian (Farsi)
  - Chinese
- Responsive UI
- Animated main menu
- Multiple levels with progression

---

## Gameplay Mechanics

### Time Energy
Collect crystals to restore time energy and use special abilities.

### Time Trails
Activate time view to reveal previous movement paths.

### Time Knots
Create temporal knots to manipulate the environment and generate unique effects.

### Ghosts
Summon previous versions of yourself to assist in solving challenges.

### Boss Fight
Face the Time Boss in the final stage.

# 🇨🇳 中文

## 项目简介

**ChronoWeaver** 是一款基于 **Python** 和 **PyQt6** 开发的二维时间操控平台游戏。

玩家可以操控时间、创建时间结点、查看时间轨迹、召唤过去的自己，并通过时间能力解决各种挑战。

### 主要功能

- 时间操控系统
- 时间轨迹可视化
- 时间结点机制
- 幽灵回放系统
- 多种敌人类型
- Boss 战斗
- 粒子特效系统
- 深色与浅色主题
- 多语言支持
  - English
  - فارسی
  - 中文
- 动态主菜单
- 多关卡设计
